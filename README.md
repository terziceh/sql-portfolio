# SQL Portfolio – Ethan Terzic

This repository showcases SQL-based projects demonstrating database design,
data modeling, data quality validation, and analytical querying across
real-world use cases.

## Projects

### 1) Database Management Project
Designed and implemented a normalized relational database including ERD design,
constraints, and analytical queries to answer business questions.

### 2) Invoice Reader – SQL Layer
SQL schema and analytics layer supporting a Python-based invoice reader that
extracts structured data from utility invoice PDFs.

### 3) ArcGIS Lightpole Database
Relational and spatial SQL supporting a geospatial asset database used to track
and analyze campus lightpole infrastructure.

## Tools
- SQL (PostgreSQL / DuckDB compatible)
- Database Design & Normalization
- Data Quality Checks
- Analytical Queries
- Geospatial SQL (PostGIS concepts)

## Notes
All data included is synthetic or redacted to preserve confidentiality.
